# 🧪 Experiment 10: CRUD Operations using Node.js + Express.js

## 📌 Objective
The objective of this experiment is to implement and demonstrate **CRUD operations (Create, Read, Update, Delete)** using a backend built with **Node.js** and **Express.js**, connected to a **MongoDB database**.  
This project showcases how to:
- Create new records in the database
- Retrieve existing records
- Update records by ID
- Delete records by ID  

The experiment also includes testing via **Browser (GET requests)** and **Postman (CRUD requests)** to validate functionality.
